# 大作业 —— 钟表

基于两种不同的设计思路实现了两个时钟版本，主要差异体现在表盘样式和交互方式上。

## 一号表盘

### 基本功能
- 基于EasyX库实现白色表盘、刻度绘制与指针转动
- 通过绘制浅色辅助直线和使用浮点数与精确PI值提高精度
- 实现表盘与表针的抗锯齿算法（详见源代码的`fixline`函数和`fixcircle`函数）

### 进阶功能
1. **数字化时间显示**
   - 在表盘上显示24小时制数字时间
   - 增强时钟的可读性和功能性

2. **鼠标交互功能**
   - 左键点击触发两种交互效果
   - 动态显示"CLICK ANYWHERE"交互提示
   - 点击后表盘产生浅金黄色渐变效果（基于`clock()`函数实现）
   - 随机位置生成三个逐渐放大的彩色圆圈（色彩和位置随机生成）

## 二号表盘

### 基本功能
- 表盘颜色和风格与一号表盘不同，其他基础功能相同

#### 特色功能
- **粉蓝渐变表盘**：基于像素级颜色处理实现从粉色到蓝色的渐变效果
- 通过多层灰度变化和颜色映射创建**金属质感**，整体色调设置为银灰色系

#### 已知问题
- **性能问题**：由于每次循环均需要逐像素绘制表盘，运行效率较低
- **视觉瑕疵**：在核心半径区域存在明显的灰色方块

#### 优化方向
考虑将表盘背景设为静态图像，仅动态更新指针部分以提升性能。
